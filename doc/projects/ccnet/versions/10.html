Version 1.8.4¶

Fixes some more issues of 1.8 as reported by users.
No breaking changes in terms of configuration or classes for custom tasks, publishers, ....

These fixes did not make it for the 1.8.3 release.

This update mainly has WebDashboard and sourcecontrol improvements, also some issues are fixed in the parallel task.

Focus points :
° Subversion/SVN 1.7 should work
° support for VS 2012
° Force button work when using the new categories in farmreport
° improve speed of several dashboard pages
° show logged in username in the dashboard
