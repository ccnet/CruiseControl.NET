Version 1.9¶

In this track we want to increase Mono Compatibility.
Console, service and dashboard must work out of the box.

Also the unit tests should work on mono, this means we must split the current test assembly into parts,
as it contains items that are Windows only.
The documentation also needs some polishing for installing and using under mono.
Initial work is here DevInfo_Mono
