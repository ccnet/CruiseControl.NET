Named Value¶

A value with an associated name. This is a generic configuration item that is used in other configuration elements.

Version¶

Available from version 1.4.4

Examples¶

1<namedValue name="The Name" value="something" />

Configuration Elements¶

Element Description Type Required Default Version
name The name of the value. String Yes n/a 1.4.4
value The actual value. String No n/a 1.4.4
