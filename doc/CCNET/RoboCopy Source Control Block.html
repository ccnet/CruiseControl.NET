RoboCopy Source Control Block

Uses RoboCopy as Source Control.

Version

Available from version 1.4.4

Examples

<sourcecontrol type="repositoryRoot">
  <repositoryRoot>C:\Somewhere</repositoryRoot>
</sourcecontrol>
