Parallel Task

Runs a set of child tasks in parallel. Each task will run at the same time as the other tasks.

To run a set of tasks in sequential order within this task, use the Sequential Task.

Version

This task is available from version 1.5.0.

Examples

Minimalist example:

<parallel>
  <tasks>
    <!-- Tasks defined here -->
  </tasks>
</parallel>

Full example:

<parallel>
  <description>Example of how to run multiple tasks in parallel.</description>
  <tasks>
    <!-- Tasks defined here -->
  </tasks>
</parallel>

Configuration Elements

Node Description Type Required Default Version
tasks The child tasks to run in parallel. Task array - see Task Blocks Yes n/a 1.5.0
description A description of this task. String No None 1.5.0
dynamicValues Any dynamic parameters to be used. See Dynamic Parameters Dynamic values array - see Dynamic Parameters No None 1.5.0

Notes

Sometimes it may be necessary to have a sequence of tasks running at the same time as a single parallel task. The Sequential Task used together with this task will achieve this.

This is an example of how this can be done:

<parallel>
  <tasks>
    <exec>
      <!-- Executable task that will run on its own -->
    </exec>
    <sequential>
      <tasks>
        <exec>
          <!-- Executable task that will run before the next task -->
        </exec>
        <exec>
          <!-- Executable task that will run after the previous task -->
        </exec>
      </tasks>
    </sequential>
  </tasks>
</parallel>
