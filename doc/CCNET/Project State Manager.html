CruiseControl.NET : Project State Manager
This page last changed on Oct 17, 2004 by mroberts.

The Project State Manager is a State Manager that uses an underlying File State Manager but automatically sets the file name to be project-specific, using the default directory.

This is an internal block and is not configured directly. Merely leave out the state configuration empty in your project to use this state manager.
