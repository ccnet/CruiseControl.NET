CruiseControl.NET : Server Log Project Plugin
This page last changed on Sep 24, 2007 by rosspatterson.

The Server Log Project Plugin shows you recent activity that has been output to the server log for a specific project. Read the Server Application Config File page for more help on build server logging.

Example

<serverLogProjectPlugin />

Configuration Elements

There is no configuration for this plugin.
