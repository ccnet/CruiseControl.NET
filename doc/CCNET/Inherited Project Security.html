Inherited Project Security

Inherit the security settings for a project from the server settings.

The security settings for a project are stored as a child element under the <project> tag.

Version

This security item is available from version 1.5.0.

Examples

<security type="inheritedProjectSecurity" />

Configuration Elements

Node Description Type Required Default Version
type The type of project security - must be inheritedProjectSecurity. String Yes n/a 1.5.0
