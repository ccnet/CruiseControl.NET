CruiseControl.NET : Using CruiseControl.NET with MSBuild
This page last changed on Oct 17, 2004 by mroberts.

Currently there is no explicit support for MSBuild in CruiseControl.NET (although such a plugin is under development). For now it is recommended that you use the Command Line Builder - see Mike Swanson's Blog at http://blogs.msdn.com/mswanson/archive/2004/10/05/238423.aspx for more help.
