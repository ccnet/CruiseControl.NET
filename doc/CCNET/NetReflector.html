CruiseControl.NET : NetReflector
This page last changed on Jun 24, 2005 by orogers.

NetReflector is the xml binding framework used by CCNet to handle the deserialisation of the configuration data into the objects used by the application. You can find out more about NetReflector on the confluence site . The NetReflector project is hosted on SourceForge . Please log any issues that you experience with NetReflector in Jira .
