CruiseControl.NET : Labeller Blocks
This page last changed on May 19, 2005 by orogers.

Labellers are used to generate the label that CCNet uses to identify the specific build. The label generated by CCNet can be used to version your assemblies or label your version control system with each build.

Types of Labeller Block are:

  • Assembly Version Labeller
  • Date Labeller
  • Default Labeller
  • File Labeller
  • Iteration Labeller
  • Last Change Labeller
  • Remote Project Labeller
  • State File Labeller
