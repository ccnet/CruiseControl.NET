CruiseControl.NET : File State Manager
This page last changed on Oct 17, 2004 by mroberts.

The File State Manager is a State Manager that saves the state for one project to a file.

Examples

Minimalist example: 

<state type="state" />

Full example: 
<state type="state" filename="mystate.state" directory="C:\CCNetState" />

Configuration Elements:

Node Description Type Required Default
filename The name of the state file to use string false ccnet.state
directory The directory to save the state file to string false The directory CCNet was launched from

Even though both of these settings are defaultable, it is recommended that you use a Project State Manager if you don't care about settings, the reason being that Project State Manager uses valid defaults for Multiple Project configurations.
