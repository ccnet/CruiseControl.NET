And Condition

Checks that all the child condition pass.

Version

Available from version 1.6

Examples

Basic example
<andCondition>
  <conditions>
    <!-- Conditions to check -->
  </conditions>
</andCondition>
Example in context
<conditional>
  <conditions>
    <andCondition>
      <conditions>
        <!-- Conditions to check -->
      </conditions>
    </andCondition>
  </conditions>
  <tasks>
    <!-- Tasks to perform if condition passed -->
  </tasks>
  <elseTasks>
    <!-- Tasks to perform if condition failed -->
  </elseTasks>
</conditional>

Configuration Elements

Element Description Type Required Default Version
conditions The conditions to check. Task Condition Blocks array Yes n/a 1.6
description A description of the condition. String No none 1.6
