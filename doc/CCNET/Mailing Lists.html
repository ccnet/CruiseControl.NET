CruiseControl.NET : Mailing Lists
This page last changed on Oct 11, 2006 by gfeldman.

We use Google Groups to host public mailing lists. To subscribe to lists, or read their archives, click the appropriate link below. You will need a (free) Google account to subscribe to these lists, and you must subscribe if you wish to post to the lists.

All community mailing lists have been moved from SourceForge to Google Groups. If you were previously subscribed to the SourceForge lists, please move your subscription to the new Google Group list. You can unsubscribe from the ccnet-user SourceForge mailing list by following the link here.

We have 3 primary mailing lists:

ccnet-user - A general purpose mailing lists for support, announcements, usage discussions, etc. Use ccnet-user if you need help using CruiseControl.NET.

ccnet-devel - This is where we discuss all things to do with the continuing development of CCNet.

ccnet-checkins - All SVN commits to the CCNet project tree are logged here.
