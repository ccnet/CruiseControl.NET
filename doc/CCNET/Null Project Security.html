Null Project Security

This element turns off security at the project level. A project with this security will give all rights to all users.

Version

This security item is available from version 1.5.0.

Examples

<security type="nullProjectSecurity"/>

Configuration Elements

Node Description Type Required Default Version
type The type of project security to use - must be nullProjectSecurity. String Yes n/a 1.5.0
