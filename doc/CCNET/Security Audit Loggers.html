Security Audit Loggers

A security audit logger will log audit events to a log.

There can be multiple audit loggers for a server, but there can only be one primary audit log. The primary audit log is the log that is read and used for retrieving records for a client. See Security Audit Readers for further details.

The following audit loggers are available:
