In Memory Security Cache

Provides an in-memory cache for sessions.

This cache stores sessions in the current instance of CruiseControl.Net. When CruiseControl.Net is restarted the sessions will be lost.

Version

This security item is available from version 1.5.0.

Examples

<cache type="inMemoryCache" duration="10" mode="sliding"/>

Configuration Elements

Node Description Type Required Default Version
type The type of caching to use - must be inMemoryCache. String Yes n/a 1.5.0
duration The duration, in minutes, that a session is stored for. Integer No 10 1.5.0
mode The type of expiration period to use. Options are either Sliding (the expiry time is moved every time a security request is made) or Fixed (expiry time never changes). String:
  • Sliding
  • Fixed
 No Sliding 1.5.0
