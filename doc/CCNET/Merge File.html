Merge File

Details on a file to merge.

Version

Available from version 1.0

Examples

Minimalist Example
<file>
  <!-- path to file -->
</file>
Full Example
<file action="Copy">
  <path>
    <!-- path to file -->
  </path>
</file>

Configuration Elements

Element Description Type Required Default Version
path The name of the file to merge. String Yes n/a 1.0
action The type of the file to merge. String - one of:
  • Merge
  • Copy
  • CData
 No Merge 1.5
