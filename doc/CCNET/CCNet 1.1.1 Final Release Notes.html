Highlights of the CCNet 1.1.1 release

The 1.1.1 release is a point release for CCNet 1.1. It contains a few bug fixes and some enhancements. See the list below for more details.

Upgrading from CCNet 1.1

CCNet 1.1.1 should be fully backwards compatible with the 1.1 release.

Upgrading directly from 1.0 to 1.1.1

If you're still using 1.0, you can go directly to 1.1.1, but there are a number of issues and incompatibilities. Please see the CCNet 1.1 Final Release Notes for details.
