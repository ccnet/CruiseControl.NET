Simple Security Plugin

Allows the user to send a user name and password to the server for authentication.

Version

Available from version 1.5

Examples

Minimalist Example
<simpleSecurity />
Full Example
<simpleSecurity hidePassword="true" />

Configuration Elements

Element Description Type Required Default Version
hidePassword Whether to hide the password field or not. Boolean No false 1.5
