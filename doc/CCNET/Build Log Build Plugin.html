CruiseControl.NET : Build Log Build Plugin
This page last changed on Jun 08, 2005 by mroberts.

The Build Log Build Plugin shows you the XML log for a specific build. Initially it is viewed inline in an HTML view, but there is also an option to download the raw XML form.

This is a Project Configurable Build Plugin.

Example

<buildLogBuildPlugin />

Configuration Elements

There is no configuration for this plugin.
