Highlights of the CCNet 1.4.4 SP1 release

This is mainly an update to the 1.4.4 release to address the issue of the wrong NetReflector version.
That gave problems with the existing plugins, also the msbuild logger problem with the hot-swap has been fixed.

Release Notes - CruiseControl .NET - Version 1.4.4 SP1

Bug

  • CCNET\-1477 - Can not start service (ccservice.exe 1.4.4.27) NetReflector wrong version (1.0.0.120)
  • CCNET\-1541 - NetReflector assembly has wrong public key token
  • CCNET\-1549 - Hot swap gives problems with default msbuild logger
