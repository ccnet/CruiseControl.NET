CruiseControl.NET : RSSFeedProjectPlugin
This page last changed on Feb 01, 2008 by rosspatterson.

This publisher generates an RSS feed reporting the builds for a project.
The project must use the RSS Publisher to produce the data for the feed.

This is available from build 1.3.0.3011 onwards.

Minimal Configuration 

<projectPlugins>
    <RSSFeed/>
 </projectPlugins>

Full Configuration

<projectPlugins>
    <RSSFeed/>
 </projectPlugins>
